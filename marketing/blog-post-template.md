## Template Structure

### 1. SEO Title (55-60 chars)
```
[Number] [Problem/Question] — [Benefit] ([Year])
Example: How Much Is a Full Bike Service in the UK? — Transparent 2026 Prices
```

### 2. Meta Description (155 chars)
```
[Question hook]. [What the article covers]. [Why reader should trust BC].
Example: Confused about bike service costs? Here's exactly what you should 
be paying for every service level in 2026 — and why London prices differ.
```

### 3. Intro (100 words max)
- Hook: Start with a relatable frustration or stat
- Acknowledge the problem
- Promise: "By the end of this guide, you'll know..."
- Brief author intro (Freddie — 10+ years experience)

### 4. Table of Contents
```
- Overview of the problem
- Section 2
- Section 3  
- Section 4
- How BikeClinique can help
- FAQ
```

## Publishing Checklist

□ Keyword in: title, URL, first 100 words, 1 H2
□ Meta description written (155 chars)
□ Internal links: 2 minimum
□ External links: 3 authoritative sources
□ Images: alt text on every image
□ CTA: clear next step
□ 301 redirect from old URL if republishing
□ Sitemap updated
□ GSC URL inspection → request indexing
